## 项目简介
仿照微信的页面实现的简单demo

## 项目技术
开发平台：vscode（自带模拟器）
框架：flutter
flutter使用插件
```
flutter_webview_plugin: "^0.2.1+2"
date_format: "^1.0.4"
```

## 主要文件
assets 主要图片处理
loading.dart 进入时的页面
app.dart 主程序页面
main.dart 项目程序路口
pubspec.yaml 项目包管理及版本设置
.vscode vscode运行下避免报错的配置
